# NFR Requirements - Unit 5: 관리자 대시보드

## 상속: Unit 0 공통 NFR 전체 적용

## Unit 5 특화 요구사항

### 성능
- 대시보드 초기 로딩: 1초 이내
- SSE 이벤트 반영: 2초 이내 (Unit 0 기준 상속)
- 최대 테이블 수: 30개 (소규모 매장 기준)
- 주문 상태 변경 API 응답: 500ms 이내
- 과거 내역 조회 API 응답: 1초 이내

### SSE 연결 관리
- 연결 끊김 시 즉시 재연결 (1초)
- 최대 재연결 시도: 무제한 (연결될 때까지)
- 연결 상태 UI 표시 (연결됨/재연결 중)

### 보안
- 모든 API에 JWT 인증 필수 (Unit 0 SecurityConfig 상속)
- 주문 삭제/상태 변경/이용 완료: 관리자 권한 검증
- 과거 내역: 해당 매장 데이터만 접근 가능

### 사용성
- 신규 주문 시각적 강조: 3초간 하이라이트 애니메이션
- 확인 팝업: 주문 삭제, 이용 완료 시 필수
- 성공/실패 토스트 메시지 표시
- 반응형 그리드: 화면 크기에 따라 카드 열 수 조정
