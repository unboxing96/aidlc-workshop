| Unit | 의존 대상 | 의존 유형 | 설명 |
|------|----------|----------|------|
| Unit 1: 테이블/인증 | (없음) | - | 독립, 다른 Unit의 기반 |
| Unit 2: 메뉴 | (없음) | - | 독립, 메뉴 데이터 제공 |

## 실행 순서

| 순서 | Unit | 근거 |
|------|------|------|
| 1 | Unit 1: 테이블/인증 | 독립, 인증 기반 제공 |
| 2 | Unit 2: 메뉴 | 독립, 메뉴 데이터 제공 |
| 3 | Unit 3: 장바구니/주문 | Unit 1, 2의 데이터 모델에 의존 |
| 4 | Unit 4: SSE | Unit 1, 3의 이벤트 소스에 의존 |
| 5 | Unit 5: 대시보드 | Unit 1, 3, 4의 API/SSE에 의존 |

**참고**: Unit 1과 Unit 2는 독립적이므로 병렬 작업 가능.
